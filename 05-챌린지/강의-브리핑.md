# SPEC 디자인팀 강의 브리핑

> 이 문서는 SPEC 디자인팀 리더가 팀원 6명에게 디자인 교육을 진행하기 위한 강의 가이드입니다.
> 총 3회 강의, 각 1.5~2시간 분량. 강의 전날 이 문서를 한 번 읽고 가세요.

---

## 전체 강의 개요

| 회차 | 주제 | 시간 | 다루는 챕터 |
|------|------|------|------------|
| 강의 1 | 디자인 기초 | 1.5시간 | Ch.0~5 |
| 강의 2 | AI 도구 소개 + 실습 | 2시간 | Ch.6~8 |
| 강의 3 | 실전 워크숍 | 2시간 | 팀별 과제 |

**강의 목표**: 비전공자 팀원들이 디자인 기본 원칙을 이해하고, AI 도구를 활용해서 실제 SPEC 홍보물을 스스로 만들 수 있게 되는 것.

**강의 방식**: 이론 설명 + 라이브 데모 + 직접 실습. 강의자가 혼자 떠드는 시간을 최소화하고, 팀원들이 직접 손을 움직이는 시간을 최대화하세요.

---

## 강의 1: 디자인 기초 (1.5시간)

### 강의 목표
- CRAP 원칙(대비, 반복, 정렬, 근접)을 이해하고 실제 디자인에서 찾아낼 수 있다.
- 색상 조합과 폰트 선택의 기본 기준을 안다.
- 이미지 다루기 기초(누끼, 해상도)를 할 수 있다.

### 시간 배분

---

#### 0:00~0:10 | 아이스브레이킹 (10분)

**진행 방법:**
카카오톡 단톡방이나 슬랙에 "지금까지 만든 것 중 가장 못 만든 디자인 공유해주세요"라고 올리고, 팀원들이 올린 결과물을 화면에 띄워서 같이 웃으면서 시작하세요.

없으면 강의자 본인의 초기 디자인 실패작을 보여줘도 됩니다. 포인트는 "디자인은 처음부터 잘하는 사람이 없다"는 걸 분위기로 전달하는 것.

**왜 이렇게 시작하냐면:** 비전공자들은 디자인 강의 시작 전에 "나는 못 할 것 같다"는 심리적 장벽이 있어요. 못 만든 것을 같이 웃으면서 보면 그 장벽이 낮아집니다.

---

#### 0:10~0:30 | Ch.0: 왜 디자인인가? (20분)

**핵심 메시지 3가지:**
1. 디자인은 재능이 아니라 원칙이다.
2. 좋은 디자인은 정보를 더 빠르게 전달한다.
3. AI 도구 덕분에 비전공자도 실무급 결과물을 낼 수 있다.

**진행 방법:**
- 같은 내용의 Before/After 포스터 2장을 나란히 보여주세요. (SPEC 완성본 폴더에서 가져오거나, 인터넷에서 "디자인 before after" 검색)
- "왜 오른쪽이 더 좋아 보이나요?" 질문 던지고 팀원들 답변 받기.
- 답변을 들으면서 "그게 바로 오늘 배울 CRAP 원칙이에요"로 자연스럽게 연결.

**선택 사항 (시간 여유 있을 때):**
YouTube에서 "디자인 원칙" 또는 "17년차 디자이너" 관련 영상의 핵심 클립 2~3분 틀어주기. 강의자가 설명하는 것보다 영상이 더 설득력 있을 때가 많아요.

---

#### 0:30~0:50 | Ch.1~2: CRAP 원칙 + 위계/구도 (20분)

**CRAP 원칙 설명 순서:**

**C (Contrast, 대비)** 먼저 설명하세요. 가장 직관적으로 이해되는 원칙이라 첫 번째로 하면 흐름이 좋아요.
- "대비가 없으면 뭐가 중요한지 모른다"
- 예시: 흰 배경에 연한 회색 텍스트 vs 흰 배경에 검은 텍스트

**R (Repetition, 반복)** 두 번째.
- "같은 요소를 반복하면 통일감이 생긴다"
- 예시: 카드뉴스 5장에서 같은 폰트, 같은 색상, 같은 레이아웃 반복

**A (Alignment, 정렬)** 세 번째.
- "정렬이 안 되면 어수선해 보인다"
- 예시: 텍스트가 제각각 위치 vs 왼쪽 정렬로 통일

**P (Proximity, 근접)** 마지막.
- "관련 있는 것끼리 가까이 놓아라"
- 예시: 제목과 부제목은 붙이고, 다른 섹션과는 간격 두기

**퀴즈 (5분):**
실제 포스터 이미지 1~2장을 화면에 띄우고 "이 포스터에서 CRAP 중 뭐가 안 되어 있나요?" 질문. 팀원들이 직접 찾아보게 하세요. 정답이 없어도 됩니다. 찾는 과정 자체가 학습이에요.

**위계와 구도 (5분):**
- 위계: "제목 > 부제목 > 본문" 크기 차이로 눈의 이동 경로를 설계하는 것
- 구도: 삼분할법 간단히 소개. "피사체를 화면 중앙이 아니라 1/3 지점에 놓으면 더 역동적으로 보인다"

---

#### 0:50~1:10 | Ch.3~4: 색상 + 타이포그래피 (20분)

**색상 (10분):**

설명보다 실습 위주로 진행하세요.

1. Coolors(coolors.co) 화면 공유하면서 스페이스바 눌러서 팔레트 생성 데모
2. "마음에 드는 팔레트 나올 때까지 스페이스바 눌러보세요" 하고 팀원들 직접 해보게 하기
3. 색상 3가지 규칙 설명:
   - 주색 1개 + 보조색 1~2개 + 강조색 1개
   - 보색 조합은 강렬하지만 피로감 줄 수 있음
   - 처음엔 무채색(흰/검/회) + 포인트 컬러 1개로 시작하는 게 안전

**타이포그래피 (10분):**

폰트 실습을 직접 해보는 게 핵심이에요.

1. 눈누(noonnu.cc) 화면 공유
2. 부록 A의 추천 폰트 7종 중 2~3개 직접 보여주기
3. "지금 눈누 들어가서 마음에 드는 폰트 하나 다운받아보세요" (5분)
4. 다운받은 폰트를 Figma나 미리캔버스에 적용해보기

폰트 조합 팁 한 가지만 강조하세요: "같은 폰트 패밀리에서 굵기만 다르게 쓰는 게 가장 안전하다."

---

#### 1:10~1:20 | Ch.5: 이미지 다루기 (10분)

**누끼 따기 라이브 데모 (7분):**
1. 구글에서 아무 사진이나 저장
2. PhotoRoom(photoroom.com/ko) 접속
3. 사진 업로드 후 배경 제거 과정 화면 공유로 보여주기
4. "이거 예전엔 포토샵으로 30분 걸렸는데 지금은 5초예요"

**해상도 설명 (3분):**
- 인쇄물: 300 DPI 이상 필요. 낮으면 흐릿하게 나옴.
- 화면용: 72~96 PPI면 충분.
- 실용적 팁: "인터넷에서 이미지 저장할 때 가능한 큰 사이즈로 저장하세요."

---

#### 1:20~1:30 | Q&A + 다음 시간 예고 (10분)

**Q&A 진행 방법:**
"지금까지 배운 것 중 헷갈리는 거 있으면 뭐든 물어보세요"로 시작. 질문이 없으면 강의자가 먼저 "제가 처음 배울 때 가장 헷갈렸던 건..." 하고 자주 나오는 질문을 먼저 꺼내세요.

**다음 시간 예고:**
"다음 시간엔 AI 도구 실습이에요. 미리캔버스, Canva, Ideogram, Gamma 직접 써볼 거예요. 각 도구에 계정 미리 만들어오세요."

계정 만들어올 도구 목록을 단톡방에 공유하세요:
- 미리캔버스 (miricanvas.com)
- Canva (canva.com)
- Ideogram (ideogram.ai)
- Gamma (gamma.app)

---

### 강의 1 준비물 체크리스트

- [ ] Before/After 포스터 예시 이미지 2~3쌍 준비
- [ ] CRAP 원칙 퀴즈용 포스터 이미지 1~2장 준비 (SPEC 레퍼런스 폴더에서)
- [ ] 눈누 사이트 미리 열어두기
- [ ] PhotoRoom 사이트 미리 열어두기
- [ ] Coolors 사이트 미리 열어두기
- [ ] 화면 공유 테스트 (강의 전 5분)
- [ ] 팀원들에게 노트북 지참 안내

## 강의 2: AI 도구 소개 + 실습 (2시간)

### 강의 목표
- 미리캔버스, Canva, Ideogram, Gamma 4가지 도구를 직접 써볼 수 있다.
- 각 도구의 장단점을 알고 상황에 맞게 선택할 수 있다.
- 실습 과제(카드뉴스 1장)를 완성할 수 있다.

### 시간 배분

---

#### 0:00~0:10 | 복습 퀴즈 (10분)

**진행 방법:**
슬라이드나 화면에 포스터 이미지 1장을 띄우고 "CRAP 원칙 중 잘 된 것과 안 된 것을 각각 하나씩 찾아보세요"라고 하세요.

팀원들이 답변하면 "맞아요, 그게 바로 A(정렬)예요" 식으로 지난 시간 내용을 자연스럽게 복습.

5분 이상 쓰지 마세요. 복습은 워밍업이지 메인이 아닙니다.

---

#### 0:10~0:30 | Ch.6: AI 도구 선택 가이드 (20분)

**핵심 메시지:**
"도구가 많아서 뭘 써야 할지 모르겠다는 분들을 위해, 상황별로 뭘 쓰면 되는지 정리했어요."

**상황별 도구 추천 (이걸 화면에 띄워두고 설명하세요):**

| 만들고 싶은 것 | 추천 도구 |
|--------------|----------|
| 카드뉴스 (한국 감성) | 미리캔버스 |
| 포스터, 배너 (다양한 템플릿) | Canva |
| 한글 텍스트 포함 이미지 | Ideogram |
| 발표자료 (PPT) | Gamma |
| 로고 | Logo.com |
| 배경 이미지 | 나노바나나 |
| 누끼 따기 | PhotoRoom |

**강조할 포인트:**
- "하나만 잘 써도 충분해요. 처음엔 미리캔버스 하나만 파세요."
- "도구는 수단이에요. 디자인 원칙을 알면 어떤 도구든 잘 쓸 수 있어요."

---

#### 0:30~0:50 | 미리캔버스 실습 (20분)

**실습 목표:** 카드뉴스 1장 완성하기

**진행 순서:**

1. **강의자 라이브 데모 (7분)**
   - 미리캔버스 접속 후 "새 디자인 만들기" 클릭
   - "카드뉴스" 템플릿 검색
   - 마음에 드는 템플릿 선택
   - 텍스트 수정, 색상 변경, 이미지 교체 과정 화면 공유로 보여주기
   - "SPEC 5기 리크루팅" 내용으로 텍스트 바꾸는 것까지 데모

2. **팀원 따라하기 (10분)**
   - "지금 여러분도 똑같이 해보세요"
   - 막히는 사람 옆에서 도와주기
   - 완전히 완성 안 해도 됨. 기본 흐름 파악이 목표.

3. **결과 공유 (3분)**
   - 1~2명 화면 공유해서 만든 것 보여주기
   - 짧게 피드백: "이 부분 CRAP 원칙 중 정렬이 잘 됐네요" 식으로

**자주 막히는 부분 미리 알아두기:**
- 폰트가 안 보일 때: 미리캔버스는 자체 폰트 사용. 로컬 폰트 설치 불필요.
- 이미지 교체: 요소 클릭 후 "이미지 교체" 버튼 또는 드래그앤드롭
- 저장: 자동 저장됨. 별도 저장 불필요.

---

#### 0:50~1:10 | Canva 실습 (20분)

**실습 목표:** 포스터 1장 완성하기

**진행 순서:**

1. **강의자 라이브 데모 (7분)**
   - Canva 접속 후 "포스터" 검색
   - 미리캔버스와 다른 점 강조: 템플릿 수가 훨씬 많고, 영어 기반이지만 한국어 지원
   - Magic Studio 기능 간단히 소개: 배경 제거, 텍스트 생성
   - 포스터 1장 만드는 과정 데모

2. **팀원 따라하기 (10분)**
   - 미리캔버스와 같은 방식으로 진행
   - "미리캔버스랑 비교해서 어떤 게 더 편한지 느껴보세요"

3. **비교 토론 (3분)**
   - "미리캔버스 vs Canva, 어떤 게 더 좋았어요?"
   - 팀원들 의견 들으면서 각 도구의 장단점 정리

**Canva 팁:**
- 교육용 계정(학교 이메일)으로 가입하면 Pro 기능 무료
- 한국어 검색보다 영어 검색이 템플릿이 더 많이 나옴
- "Brand Kit" 기능으로 SPEC 브랜드 색상, 폰트 저장해두면 편함

---

#### 1:10~1:30 | Ideogram 실습 (20분)

**실습 목표:** 한글 텍스트가 포함된 이미지 1장 생성하기

**진행 순서:**

1. **강의자 라이브 데모 (7분)**
   - Ideogram 접속 후 프롬프트 입력창 보여주기
   - 영어 프롬프트 예시 입력: `"A modern poster with Korean text 'SPEC 5기 모집' in bold, minimalist design, blue and white color scheme"`
   - 생성 결과 보여주기
   - 한글 텍스트가 어느 정도 잘 나오는지 확인

2. **프롬프트 작성 팁 설명 (5분)**
   - 영어로 쓰는 게 결과가 훨씬 좋음
   - 스타일 키워드 추가: minimalist, modern, bold, elegant, colorful 등
   - 색상 명시: blue and white, dark background, pastel colors 등
   - 텍스트 내용은 따옴표로 감싸기

3. **팀원 실습 (8분)**
   - 각자 프롬프트 써서 이미지 생성해보기
   - 결과물 단톡방에 공유

**주의사항:**
- 무료 플랜은 하루 10장 제한. 실습 전에 미리 알려주세요.
- 한글 텍스트가 완벽하지 않을 수 있음. "AI가 한글을 완벽하게 쓰진 못하지만, 배경 이미지로 쓰고 텍스트는 따로 올리면 됩니다"라고 설명.

---

#### 1:30~1:50 | Gamma 실습 (20분)

**실습 목표:** 발표자료 3장 만들기

**진행 순서:**

1. **강의자 라이브 데모 (7분)**
   - Gamma 접속 후 "새 프레젠테이션" 클릭
   - 주제 입력: "SPEC 디자인팀 소개"
   - AI가 자동으로 슬라이드 구성 + 디자인하는 과정 보여주기
   - 생성된 슬라이드 텍스트 수정, 이미지 교체 방법 데모

2. **팀원 따라하기 (10분)**
   - 각자 주제 정해서 발표자료 만들어보기
   - 주제 예시: "내가 좋아하는 것 소개", "SPEC 활동 계획" 등

3. **결과 공유 (3분)**
   - 1~2명 결과물 화면 공유

**Gamma 팁:**

## CRAP 체크리스트 (프린트용)

> 강의 3에서 팀원들에게 나눠주세요. 피드백할 때 이 체크리스트 기준으로 하면 됩니다.

---

**디자인 CRAP 체크리스트**

작업물: _________________ 작성자: _________________

**C - Contrast (대비)**
- [ ] 제목과 배경 사이에 충분한 명도 차이가 있다
- [ ] 중요한 정보가 덜 중요한 정보보다 눈에 잘 띈다
- [ ] 텍스트가 배경과 구분되어 읽기 편하다

**R - Repetition (반복)**
- [ ] 같은 종류의 요소는 같은 폰트/색상/크기를 사용했다
- [ ] 디자인 전체에 일관된 스타일이 있다
- [ ] 여러 장이라면 각 장의 스타일이 통일되어 있다

**A - Alignment (정렬)**
- [ ] 텍스트와 이미지가 일정한 기준선에 맞춰 정렬되어 있다
- [ ] 요소들이 제각각 위치하지 않고 가지런하다
- [ ] 중앙 정렬, 왼쪽 정렬 등 정렬 방식이 일관되다

**P - Proximity (근접)**
- [ ] 관련 있는 정보끼리 가까이 배치되어 있다
- [ ] 다른 섹션과는 충분한 간격이 있다
- [ ] 여백이 적절해서 답답하지 않다

개선할 점: _________________________________
