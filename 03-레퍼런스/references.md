# 개요

> **디자인 작업 시 참고할 수 있는 레퍼런스 사이트와 자료를 모아둔 섹션입니다.** 영감이 필요하거나, 폰트/색상/로고 레퍼런스를 찾을 때 활용합니다.
>
> 카테고리별 레퍼런스 목록은 다음과 같습니다.
> 1. 디자인 영감 사이트
> 2. 폰트 레퍼런스
> 3. 색상 레퍼런스
> 4. 로고 레퍼런스
> 5. 템플릿 모음


# 디자인 영감

> 레이아웃 아이디어가 막힐 때 빠르게 참고할 수 있는 사이트 모음입니다. 이미지 소스, 이모지, 커뮤니티 기반 에셋을 함께 활용하면 작업 속도와 완성도를 동시에 높일 수 있습니다.

1. [Unsplash](https://unsplash.com/ko) — 무료 고퀄리티 이미지를 검색할 수 있는 대표 사이트입니다.
2. [Emojigraph (Apple)](https://emojigraph.org/apple/) — 애플 이모지를 복사해 디자인 요소로 바로 활용할 수 있습니다.
3. Figma Community (검색어: `3d icon`) — 피그마 커뮤니티에서 3D 에셋 파일을 찾아 복사·붙여넣기로 사용할 수 있습니다.
4. [Dribbble](https://dribbble.com/) — 최신 UI/브랜딩/그래픽 트렌드를 빠르게 확인하기 좋은 레퍼런스 플랫폼입니다.
5. [Behance](https://www.behance.net/) — 실무 포트폴리오 중심으로 완성도 높은 작업 사례를 참고할 수 있습니다.

# 폰트 레퍼런스

> 인스타그램 템플릿 작업 기준으로, 제목은 시선을 끌 수 있는 굵은 폰트를, 본문은 모바일 가독성이 높은 고딕 계열 폰트를 사용하는 것을 권장합니다. 장식형 제목 폰트는 포인트 단어에 제한적으로 사용하면 통일감을 유지하기 좋습니다.

1. [눈누](https://noonnu.cc/) — 무료 한글 폰트를 모아둔 사이트로, 폰트 탐색과 다운로드를 한 번에 할 수 있습니다.
2. [G마켓 산스](https://noonnu.cc/font_page/366) — 제목용으로 적합한 볼드 계열 폰트입니다.
3. [강원교육튼튼체](https://noonnu.cc/font_page/805) — 제목용으로 사용하기 좋고, 강조 표현에도 잘 어울립니다.
4. [에스코어 드림](https://noonnu.cc/font_page/223) — 다양한 두께를 제공해 제목 계층을 만들기 좋습니다.
5. [평창평화체](https://noonnu.cc/font_page/950) — 장식 제목용으로 EVENT, SALE 같은 포인트 단어에 적합합니다.
6. [창원단감아삭체](https://noonnu.cc/font_page/731) — 장식 제목용 서브 폰트로 활용하기 좋습니다.
7. [프리텐다드](https://noonnu.cc/font_page/694) — 본문용으로 가장 안정적인 가독성을 제공하는 폰트입니다.
8. [본고딕(노토산스)](https://noonnu.cc/font_page/34) — 본문용 고딕 계열 대안으로 활용하기 좋습니다.

# 색상 레퍼런스

> 기본 브랜드 팔레트를 먼저 고정하고, 이미지에서 색을 추출해 보조색을 맞추면 카드뉴스 톤앤매너를 안정적으로 유지할 수 있습니다.

1. SPEC 공식 색상 팔레트 — 브랜드 기본 컬러로 우선 적용합니다.
   - Primary: `#FF6C0F`
   - Variant: `#FF9900`
   - Secondary: `#4C4780`
   - Black: `#202020`
   - White: `#FCFCF8`
2. 피그마 스포이드 활용 팁 — 배경/이미지에서 색을 바로 뽑아 쓸 때는 `I` 키(스포이드)를 눌러 원하는 지점의 색을 선택하면 빠르게 색 조합을 맞출 수 있습니다.
3. [Coolors](https://coolors.co/) — 빠르게 팔레트를 생성하고 조합을 탐색할 수 있습니다.
4. [Adobe Color](https://color.adobe.com/) — 색상환 기반으로 조화로운 팔레트를 설계할 수 있습니다.
5. [Color Hunt](https://colorhunt.co/) — 트렌디한 팔레트를 레퍼런스로 확인하기 좋습니다.

# 로고 레퍼런스

> 로고 스타일을 잡을 때 참고하는 레퍼런스 모음입니다. 산세리프/세리프 로고 분위기부터 AI 로고 생성, 실제 포트폴리오 사례까지 함께 봅니다.

1. [LogoAI](https://www.logoai.com/logo/) — AI 로고 생성 시안 참고
2. [Brandmark Templates](https://chat.brandmark.io/templates) — 브랜드마크 템플릿 레퍼런스
3. [Ideogram Logo Explore](https://ideogram.ai/t/explore?f=logo) — AI 로고 생성 결과물 참고
4. [Behance Logofolio](https://www.behance.net/search/projects/logofolio?tracking_source=typeahead_search_suggestion) — 로고 포트폴리오 모음
5. [Behance Logo](https://www.behance.net/search/projects/logo?tracking_source=typeahead_search_direct) — 글로벌 로고 작업물
6. [Behance 로고](https://www.behance.net/search/projects/%EB%A1%9C%EA%B3%A0?tracking_source=typeahead_search_direct) — 한글 검색용 로고 작업물
7. [Pinterest Sans Serif Logo](https://kr.pinterest.com/search/pins/?q=sanserif%20logo&rs=typed) — 고딕/산세리프 로고 참고
8. [Pinterest Serif Logo](https://kr.pinterest.com/search/pins/?q=serif%20logo&rs=typed) — 명조/세리프 로고 참고
9. [Fonts In Use](https://fontsinuse.com/?page=1) — 실제 타이포 사용 사례 확인
