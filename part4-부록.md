### 폰트 설치 방법

1. 위 링크 접속 → "다운로드" 클릭
2. 압축 풀기
3. `.ttf` 또는 `.otf` 파일 더블클릭 → "설치"
4. Figma, 미리캔버스 등에서 바로 사용 가능

> Figma 웹 버전은 **Figma Font Helper** 앱 설치 필요. 설치 후 브라우저 재시작.

## 부록 D. 디자인 용어 사전

> 자주 쓰는 순으로 정렬. 비전공자 기준으로 풀어씀.

| 용어 | 원어 | 쉬운 설명 |
|------|------|-----------|
| **레이아웃** | Layout | 뭘 어디에 놓을지 결정하는 것. 레이아웃이 좋으면 보기 편하고, 나쁘면 어수선해 보임 |
| **위계** | Hierarchy | 눈이 가는 순서 설계. 제목 크게, 부제목 중간, 본문 작게가 기본 |
| **타이포그래피** | Typography | 글자 디자인 전반. 폰트 고르고, 크기 정하고, 간격 맞추는 것 |
| **여백** | White Space | 아무것도 없는 빈 공간. 여백이 없으면 답답하고, 적당히 있으면 세련돼 보임 |
| **컬러 팔레트** | Color Palette | 디자인에 쓸 색 묶음. 보통 3~5색. 주색+보조색+강조색으로 구성 |
| **그리드** | Grid | 보이지 않는 정렬선. Figma에서 Cmd+' 누르면 표시됨 |
| **자간** | Tracking / Kerning | 글자 사이 간격. 제목은 약간 좁게, 본문은 기본값 유지 |
| **행간** | Leading | 줄과 줄 사이 간격. 본문은 폰트 크기의 1.5배가 기본 |
| **톤앤매너** | Tone & Manner | 디자인의 분위기와 성격. 색상, 폰트, 이미지 스타일이 일관되어야 함 |
| **누끼** | Cutout | 배경 제거. PhotoRoom, Canva 등에서 AI로 자동 처리 가능 |
| **목업** | Mockup | 실제 환경에 디자인을 합성한 미리보기. 포스터를 벽에 붙인 것처럼 보여주는 이미지 |
| **브랜딩** | Branding | 로고, 색상, 폰트 등 브랜드의 시각적 정체성 전체 |
| **해상도** | Resolution | 이미지 선명도. 인쇄용 300 DPI 이상, 화면용 72~96 PPI |
| **벡터** | Vector | 수학 공식으로 그린 그림. 아무리 확대해도 안 깨짐. 파일 형식: SVG, AI, EPS |
| **래스터** | Raster | 픽셀로 이루어진 그림. 확대하면 깨짐. 사진이 대표적. 파일 형식: JPG, PNG |
| **프레임** | Frame | Figma에서 디자인 영역. 포스터 1장 = 프레임 1개 |
| **컴포넌트** | Component | 재사용 가능한 디자인 조각. 원본 수정하면 복사본도 자동으로 바뀜 |
| **블리드** | Bleed | 인쇄 재단선 바깥 여유분. 보통 3mm. 재단 후 흰 여백 생기지 않도록 배경을 조금 더 크게 만드는 것 |
| **크롭** | Crop | 이미지 자르기. 필요한 부분만 남기고 나머지를 잘라내는 것 |
| **프롬프트** | Prompt | AI에게 내리는 지시. "이런 이미지 만들어줘" 하고 입력하는 텍스트. 잘 쓸수록 결과물이 좋아짐 |
