# 제목을 여기에 입력

> **이 글에 대한 간단한 소개를 여기에 작성합니다.** 이 글이 무엇에 대한 글인지, 왜 읽어야 하는지를 1~2문장으로 설명합니다. 읽는 사람이 이 글을 계속 읽을지 말지 판단하는 데 도움이 됩니다.
>
> 이 챕터에서 배우실 내용은 다음과 같습니다.
> 1. 첫 번째 핵심 내용
> 2. 두 번째 핵심 내용
> 3. 세 번째 핵심 내용

## 소제목 1

본문을 여기에 작성합니다. 18px, 160% 행간으로 표시됩니다.

문단 안에서 **이렇게 강조**하면 주황색 볼드로 표시됩니다.

***매우 강조***는 주황색 볼드로 표시됩니다. 글의 핵심 부분에 사용합니다.

1. **리스트 안에서 강조**하면 흰색 볼드로 표시됩니다.
2. 리스트 항목의 제목이나 키워드에 활용합니다.

## 작성 규칙

| 마크다운 | 결과 | 용도 |
|---------|------|------|
| `# 제목` | 42px 흰색 굵게 | 페이지 제목 (1개만) |
| `> 인용문 + 넘버링` | 주황 보더 + 연한 주황 배경 | **항상 제목 바로 아래에** |
| `## 소제목` | 30px 흰색 굵게 + 구분선 | 큰 섹션 구분 |
| `### 하위 소제목` | 19px 주황색 + 구분선 | 세부 항목 구분 |
| `**텍스트**` (문단 안) | 주황색 볼드 | 문단 내 강조 |
| `**텍스트**` (리스트 안) | 흰색 볼드 | 리스트 항목 강조 |
| `***텍스트***` | 주황색 볼드+이탤릭 | 매우 강조 (핵심) |
| `` `코드` `` | 클릭시 복사 | 색상코드, 명령어 |
| `![설명](파일.png)` | 반응형 이미지 + 캡션 | 사진 첨부 |
| `[링크](url)` | 새 탭에서 열림 | 외부 링크 |
| `---` | 구분선 | 섹션 구분 |
