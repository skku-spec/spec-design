## 바이브 코딩이란?

바이브 코딩(Vibe Coding)은 AI에게 자연어로 지시하여 코드를 작성하는 새로운 개발 방식입니다. 프롬프트만으로 웹사이트, 자동화 스크립트, 마케팅 도구를 만들 수 있습니다.

### 왜 바이브 코딩인가?

1. **코딩 경험 불필요** — AI가 코드를 작성하고, 우리는 방향을 잡아줍니다.
2. **빠른 프로토타이핑** — 아이디어를 몇 분 만에 동작하는 결과물로 만들 수 있습니다.
3. **마케팅 자동화** — 반복 작업을 자동화하는 도구를 직접 만들 수 있습니다.

## 환경 세팅

환경 세팅 방법은 **Claude Code** 챕터에서 자세히 다룹니다. 요약하면 다음과 같습니다.

1. **Node.js 설치** — [nodejs.org](https://nodejs.org)에서 LTS 버전 다운로드
2. **Claude CLI 설치** — 터미널에서 `npm install -g @anthropic-ai/claude-cli` 입력
3. **Claude 계정** — [anthropic.com](https://anthropic.com)에서 가입 후 Pro 플랜 이상 결제
4. **프로젝트 폴더 생성** — `mkdir 프로젝트이름` → `cd 프로젝트이름` → `claude`

## 프로젝트 목록

난이도 순서대로 진행하는 것을 추천합니다.

1. **자기소개 페이지** — 가장 기본적인 웹페이지. 이름, 사진, 간단한 소개글.
2. **D-day 계산기** — 날짜 입력하면 남은 일수 표시. 입력/출력 기초.
3. **할 일 목록** — 항목 추가/삭제 기능. 버튼 동작 익히기.
4. **이벤트 랜딩페이지** — 동아리 행사 안내 페이지. 실전 레이아웃 연습.
