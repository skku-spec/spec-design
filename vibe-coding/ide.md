## IDE가 뭔가요?

IDE는 **Integrated Development Environment**, 한국어로 **통합 개발 환경**입니다. 쉽게 말하면 **코드를 쓰는 전용 프로그램**입니다.

한글 문서를 쓸 때 한글(HWP)이나 워드를 쓰고, 발표자료를 만들 때 파워포인트를 쓰는 것처럼, 코드를 쓸 때는 IDE를 사용합니다. 메모장에도 코드를 적을 수 있지만, IDE를 쓰면 다음과 같은 기능을 제공받을 수 있습니다.

1. **코드 색상 구분** — 코드의 종류에 따라 색이 자동으로 칠해져 읽기 편합니다.
2. **자동 완성** — 몇 글자만 입력하면 나머지를 자동으로 추천해 줍니다.
3. **오류 표시** — 틀린 부분에 빨간 밑줄이 나타납니다. 맞춤법 검사기와 비슷합니다.
4. **파일 관리** — 프로젝트의 폴더와 파일을 한눈에 볼 수 있습니다.
5. **터미널 내장** — 프로그램을 실행하거나 명령어를 입력하는 창이 내장되어 있습니다.

### 바이브 코딩에서 IDE가 중요한 이유

바이브 코딩은 AI에게 "이런 거 만들어줘"라고 말하면 AI가 코드를 작성해 주는 방식입니다. 이때 AI가 작성한 코드가 **어디에 저장되는지**, 그리고 **결과를 바로 확인할 수 있는 곳**이 바로 IDE입니다.

IDE 안에 AI 기능이 내장되어 있으면, 대화하듯 지시하고 → 코드가 자동 생성되고 → 바로 실행해서 결과를 보는 흐름이 한 화면 안에서 이루어집니다. 이 흐름이 바이브 코딩의 핵심입니다.

## VS Code

**공식 사이트**: https://code.visualstudio.com/

**가격**: 무료

![VS Code 메인 페이지](vibe/vscode-main.png)

VS Code(Visual Studio Code)는 마이크로소프트가 만든 무료 코드 편집기입니다. 전 세계에서 가장 많이 쓰이는 IDE이며, 아래에서 소개할 Cursor와 Antigravity도 모두 VS Code를 기반으로 만들어졌습니다.

### 특징

1. **무료 + 오픈소스** — 누구나 무료로 사용할 수 있습니다.
2. **확장 프로그램(Extension)** — 마치 스마트폰에 앱을 설치하듯, 필요한 기능을 추가로 설치할 수 있습니다. AI 코딩 확장도 여기에 포함됩니다.
3. **거의 모든 언어 지원** — 파이썬, 자바스크립트, HTML 등 어떤 프로그래밍 언어든 사용할 수 있습니다.
4. **가장 큰 커뮤니티** — 궁금한 점이 있으면 검색하면 거의 다 답이 나옵니다.

### AI 코딩을 하려면?

VS Code 자체에는 AI 기능이 내장되어 있지 않습니다. 대신 **확장 프로그램**을 설치해서 AI 기능을 추가합니다.

- **GitHub Copilot** — 마이크로소프트의 공식 AI 코딩 도구 (월 $10)
- **Claude Code 확장** — Claude AI를 VS Code 안에서 사용 (설치 후 터미널에서 실행)

VS Code는 기본기이기 때문에, VS Code에 익숙해지면 Cursor나 Antigravity로 넘어가는 것이 자연스럽습니다.

## Cursor

**공식 사이트**: https://cursor.sh/

**가격**: 무료 플랜 있음 (Pro $20/월)

![Cursor 메인 페이지](vibe/cursor-main.png)

Cursor는 VS Code를 복제해서 AI 기능을 **처음부터 내장**시킨 IDE입니다. 겉모습은 VS Code와 거의 동일하지만, AI가 자연스럽게 녹아 있다는 점이 다릅니다.

### 특징

1. **AI 채팅 내장** — 오른쪽에 AI 채팅창이 있어서, 코드에 대해 질문하거나 수정을 요청할 수 있습니다.
2. **코드 자동 생성** — "로그인 페이지 만들어줘"라고 입력하면 코드를 직접 생성해 줍니다.
3. **파일 인식** — 현재 열려 있는 파일과 프로젝트 구조를 AI가 이해하고 있어서, 맥락에 맞는 답변을 해줍니다.
4. **VS Code 확장 호환** — VS Code의 확장 프로그램을 그대로 설치해서 사용할 수 있습니다.

### VS Code와의 차이

| 항목 | VS Code | Cursor |
|------|---------|--------|
| AI 기능 | 확장 프로그램 설치 필요 | 내장 |
| 가격 | 무료 | 무료 플랜 + Pro $20/월 |
| 기반 | 원본 | VS Code 포크(복제) |
| 학습 곡선 | 낮음 | 낮음 (VS Code와 동일) |

### 이런 사람에게 추천

- AI 코딩을 가장 간편하게 시작하고 싶은 경우
- 설치하자마자 바로 AI와 대화하면서 코딩하고 싶은 경우

## Antigravity

**공식 사이트**: https://antigravity.dev/

**가격**: 무료

Antigravity는 VS Code 기반의 IDE로, **Claude Code를 확장 프로그램으로 설치**하여 사용합니다. 우리 SPEC 디자인팀이 현재 사용하고 있는 IDE입니다.

### 특징

1. **Claude Code 연동** — Claude AI를 IDE 안에서 바로 사용할 수 있습니다.
2. **VS Code와 동일한 인터페이스** — VS Code를 써본 적이 있다면 바로 적응할 수 있습니다.
3. **무료** — 별도 구독료 없이 사용할 수 있습니다 (Claude API 비용은 별도).

### 세팅 방법 (1주차에 진행한 내용)

1. Antigravity를 공식 사이트에서 다운로드하여 설치합니다.
2. Claude Code 확장 프로그램을 설치합니다.
3. 단축키를 설정합니다:
   - `Cmd + 2` (Mac) / `Ctrl + 2` (Windows) — Claude Code 열기
   - `Alt + E` / `Ctrl + E` / `Cmd + E` — 추가 키 바인딩
4. 퍼미션 댄저러스 모드 — AI가 매번 확인을 받지 않고 자율적으로 작업하는 모드입니다. 편리하지만, 작업물이 삭제될 수 있는 위험이 있으므로 주의가 필요합니다.

### 어떤 IDE를 써야 할까?

| 항목 | VS Code | Cursor | Antigravity |
|------|---------|--------|-------------|
| AI 내장 | ❌ | ✅ | ❌ (확장 설치) |
| Claude Code | 확장 설치 | 확장 설치 | 확장 설치 |
| 가격 | 무료 | 무료~$20/월 | 무료 |
| 추천 대상 | 기본기 학습 | AI 코딩 입문 | SPEC 팀 표준 |

**우리 팀은 Antigravity + Claude Code 조합을 사용합니다.** 1주차 OT에서 세팅을 완료했으므로, 이미 설치가 끝난 상태입니다. 아직 설치하지 않은 팀원은 1주차 챌린지 내용을 참고해 주세요.
